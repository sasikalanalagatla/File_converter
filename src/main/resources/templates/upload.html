Smart PDF Processor

• Select one PDF for conversion
• Select multiple PDFs to merge them
• Hold Ctrl (Windows) / Cmd (Mac) to choose several files at once
Note: If you select the wrong file(s), refresh the page (F5) and start over.
The browser doesn't allow removing individual files after selection.
Made with Spring Boot + Thymeleaf • Max file size: ~10MB per file (configurable)
